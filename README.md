# chibi-kingdoms-smart-contracts

Smart contracts of Chibi Clash

## Compile contracts

```bash
npx hardhat compile
```

## Test contracts

```bash
npx hardhat test
```

## Start local node

```bash
npx hardhat node
```

## Deploy smart contracts

```bash
VERIFY_CONTRACT=true npx hardhat run --network base scripts/deploy-chibi-kingdom.ts
```
